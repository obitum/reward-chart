# Minecraft Reward Chart

A Minecraft-themed reward chart for kids. Click blocks to "mine" them and track progress toward 101 blocks.

**[View the chart](https://obitum.github.io/reward-chart/)**

## Features

- 101 blocks in a 10-column grid with geological layers (grass, dirt, cobblestone, deepslate)
- Milestone ore blocks at key targets (coal, iron, emerald, gold, redstone, diamond)
- Stone-dig sound effects on each click
- TNT-animated reset with fuse sounds, flashing, and layered explosions
- Progress saved in localStorage
